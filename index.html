Tutorial

Cómo usar la app

Sigue estos pasos para empezar a registrar asistencia en minutos.

1

Crea tus materias

En la barra superior verás el selector de materia. Haz clic en + Nueva para crear tu primera materia (por ejemplo: Matemáticas, Historia, Inglés).

  • Puedes crear tantas materias como necesites.
  • Usa ✎ Renombrar para cambiar el nombre en cualquier momento.
  • El selector cambia de materia al instante — todo el contenido se actualiza.
2

Agrega alumnos

Ve a la pestaña Alumnos y escribe el nombre de cada alumno, uno por uno, o usa Importar lista para pegar varios nombres de golpe (un nombre por línea).

  • Cada materia tiene su propia lista de alumnos independiente.
  • Puedes copiar y pegar directo desde Excel o un documento de texto.
3

Registra la asistencia

Ve a la pestaña Asistencia. La app abre automáticamente en la semana actual.

  • Usa el toggle de cada alumno para marcarlo Presente o Ausente.
  • Usa ✓ Todos presentes para marcar la clase de golpe y luego ajusta los ausentes.
  • Navega a semanas anteriores o futuras con las flechas ← →.
  • Puedes editar semanas pasadas en cualquier momento.
4

Consulta el reporte

Ve a la pestaña Reporte. Selecciona la materia en el selector del reporte para ver:

  • Estadísticas globales: alumnos, semanas, % de asistencia total.
  • Tabla con el historial semana a semana por alumno.
  • Badge de color por alumno: ≥ 80% · ≥ 60% · < 60%
5

Guarda y exporta

Usa el botón ↓ Guardar del encabezado para descargar un archivo .json con todos tus datos. La próxima vez que abras la app, usa ↑ Cargar para restaurarlos.

  • El botón ↓ Exportar CSV en el reporte genera un archivo listo para Excel.
  • Si hay cambios sin guardar, el indicador muestra ● Sin guardar.
  • La app también guarda automáticamente en el navegador como respaldo.
Tus datos

¿Dónde se guarda la información?

La app usa dos niveles de persistencia para que nunca pierdas un registro.

🔄

Guardado automático (localStorage)

Cada cambio se guarda automáticamente en el navegador. Si cierras la pestaña y la vuelves a abrir, todo sigue ahí.

⚠️ Se borra si limpias los datos del navegador. Úsalo como respaldo de sesión.
📁

Archivo JSON (recomendado)

Haz clic en ↓ Guardar para descargar un archivo .json con todos tus datos. Para restaurar, usa ↑ Cargar.

✅ Permanente. No depende del navegador. Puedes usarlo en otro equipo.
📊

CSV para reportes

Desde la pestaña Reporte, exporta un .csv de cualquier materia para abrirlo en Excel o Google Sheets.

Solo lectura. No se puede volver a cargar como backup, es para compartir o archivar.
