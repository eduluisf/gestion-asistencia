Características

Todo lo que necesitas

Sin instalaciones, sin cuentas. Un archivo que abre en cualquier navegador.

  • Por materias

    Crea tantas materias como necesites. Cada una tiene su propia lista de alumnos y sus propios registros, completamente independientes.

  • Toggle semanal

    Marca presentes y ausentes con un switch. Navega libremente entre semanas pasadas y futuras para editar cuando lo necesites.

  • Reporte automático

    Visualiza el porcentaje de asistencia por alumno con indicadores de color. Estadísticas globales por materia en un vistazo.

  • Exportar a CSV

    Descarga el reporte de cualquier materia en formato CSV, compatible con Excel y Google Sheets. Con codificación UTF-8.

  • Sin perder datos

    Guarda un archivo .json con todos los datos y cárgalo cuando quieras. Tus registros viajan contigo sin depender del navegador.

  • Sin instalación

    Un solo archivo HTML. Sin servidor, sin base de datos, sin conexión a internet requerida. Abre y empieza a trabajar.

Tutorial

Cómo usar la app

Cinco pasos para empezar a registrar asistencia en minutos.

  1. 1

    Crea tus materias

    En la barra superior está el selector de materia. Usa el botón + Nueva para crear la primera (por ejemplo: Matemáticas, Historia, Inglés).

    • Puedes crear todas las materias que necesites.
    • Usa ✎ Renombrar para cambiar el nombre en cualquier momento.
    • El selector cambia de materia al instante y todo el contenido se actualiza.
  2. 2

    Agrega alumnos

    En la pestaña Alumnos escribe un nombre y presiona Enter, o usa Importar lista para pegar varios nombres de golpe, uno por línea.

    • Cada materia tiene su propia lista de alumnos independiente.
    • Puedes copiar y pegar directamente desde Excel o un documento de texto.
  3. 3

    Registra la asistencia

    En la pestaña Asistencia la app abre en la semana actual. Usa el toggle de cada alumno para marcarlo Presente o Ausente.

    • Usa Todos presentes para marcar la clase rápido y luego ajusta los ausentes.
    • Navega a semanas anteriores o futuras con las flechas o el selector de fecha.
    • Puedes editar semanas pasadas en cualquier momento.
  4. 4

    Consulta el reporte

    En la pestaña Reporte selecciona la materia para ver estadísticas globales y el historial semana a semana de cada alumno.

    • Los porcentajes se muestran con indicadores de color: ≥ 80% ≥ 60% < 60%
    • Puedes ver el reporte de cualquier materia sin cambiar la materia activa.
  5. 5

    Guarda y exporta

    Usa el botón ↓ Guardar del encabezado para descargar un archivo .json con todos tus datos. La próxima vez usa ↑ Cargar para restaurarlos.

    • El botón ↓ Exportar CSV en el reporte genera un archivo listo para Excel.
    • Un indicador en la cabecera avisa cuando hay cambios sin guardar en archivo.

Tus datos

¿Dónde se guarda la información?

Dos niveles de persistencia para que nunca pierdas un registro.

Guardado automático

Cada cambio se guarda automáticamente en el almacenamiento local del navegador. Si cierras la pestaña y vuelves, todo sigue ahí.

Se borra si limpias los datos del navegador. Úsalo como respaldo de sesión.

Archivo JSON (recomendado)

Haz clic en ↓ Guardar para descargar un archivo con todos tus datos. Para restaurar en cualquier equipo, usa ↑ Cargar.

Permanente. No depende del navegador. Funciona en cualquier dispositivo.

Exportar CSV

Desde el reporte, exporta un .csv de cualquier materia para abrirlo en Excel o Google Sheets.

Solo para compartir o archivar. No se puede usar como backup para recargar.
